## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-portfolio.git
    cd your-portfolio
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the development server:
    ```bash
    npm run dev
    ```
4. Open `http://localhost:3000`.
